# TypeScript Minimal Starter

Verified against API ledger: 2026-05-18.

This is a minimal Copilot SDK starter, not a full application. Recheck the installed SDK before production use.

Run shape:

```bash
npm install @github/copilot-sdk
npx tsx index.ts "Say hello in one short sentence."
```

It demonstrates client/session lifecycle, a conservative permission handler, event capture, prompt sending, and cleanup.
