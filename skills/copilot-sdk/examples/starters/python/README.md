# Python Minimal Starter

Verified against API ledger: 2026-05-18.

This is a minimal Copilot SDK starter, not a full application. Recheck the installed SDK before production use.

Run shape:

```bash
pip install copilot-sdk[telemetry]
python main.py "Say hello in one short sentence."
```

It demonstrates async client/session lifecycle, a conservative permission handler, event capture, prompt sending, and cleanup.
